# 经典力学
**经典力学**，或者**理论力学**，是理论物理的开端课程。杨振宁说：“国内物理系搞了四大力学，压得学生喘不过气来。四大力学当然重要，但是它们仅是物理的骨架，没有血肉也是不能理解物理的。”

为了学会利用物理学的观点来分析处理问题，或者说建立问题的物理模型（学习新科技也有异曲同工之妙），就得对四大力学的基本概念，作出一个基本的理解，让这些东西进入自己的血液，成为自己大脑的一部分，它们才能招致即来，来之能战！
而不是一堆教条，遇到问题现翻书本，照抄教条，是没有什么用的。这种基本理解不要求全，要求实用，要突出自己的见解，不懂的可以不写！

# 理论力学（物理基地班）
## 3学分
### 成绩：98，排名：1/42
参考教材是金尚年的《理论力学》，这本书写得一般。任课老师是学工程的，导致讲课内容非常不物理。期末考试有很多莫名其妙的“思考题”，美其名曰方便捞人。
但这些“思考题”经常没把问题描述清楚，或许是出题老师也不太会物理，或许是随便在书上看到一段话就改成疑问句。
考试想及格不难，把基本的Lagrange力学和Hamilton力学搞懂就好，但想冲击高分，需要**把课本背下来**。这是我们院专业课的通病：**理解不如背诵**。
